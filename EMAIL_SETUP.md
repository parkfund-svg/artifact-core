---

## 이메일이 안 오는 이유와 해결방법

### 1단계: Apps Script 재배포 (필수!)

코드를 수정했으므로 **반드시 재배포**해야 합니다:

1. Apps Script 에디터에서 수정된 코드 붙여넣기 (backend-apps-script.js)
2. **저장** (Ctrl+S)
3. 오른쪽 상단 **배포** → **배포 관리** 클릭
4. 연필(✏️) 아이콘 클릭 (편집)
5. **버전**: "새 버전" 선택
6. **배포** 클릭
7. 새 URL이 나오면 (같을 수도 있음) 확인

### 2단계: Gmail 권한 확인

1. Apps Script 에디터에서 코드 저장 후
2. 상단의 **실행** 버튼 옆 함수 선택: `doPost` 선택
3. **실행** 클릭
4. 권한 요청이 나오면:
   - "권한 검토" 클릭
   - Google 계정 선택
   - "고급" → "안전하지 않은 페이지로 이동"
   - **허용** 클릭

### 3단계: 테스트

1. 홈페이지에서 Contact 폼 제출
2. 30초 정도 기다리기
3. parkfund@naver.com 메일함 확인
4. 스팸 메일함도 확인!

---

## 이메일이 계속 안 올 때

### Apps Script 로그 확인

1. Apps Script 에디터 왼쪽 메뉴
2. **실행** 아이콘 클릭
3. 최근 실행 내역에서 오류 확인
4. "이메일 발송 성공" 또는 "실패" 메시지 확인

### 대안: Google Sheets 알림 설정

이메일이 계속 안 되면 Google Sheets 자체 알림 사용:

1. Google Sheets 열기
2. **도구** → **알림 규칙**
3. "사용자가 변경할 때" 선택
4. 이메일 주소 입력
5. 저장

→ 새 데이터가 추가되면 자동으로 이메일 알림!

## 관리자 페이지 접속

1. **URL**: `https://artifact-core.com/admin.html`
2. **비밀번호**: `artifact2025`
3. 입력하면 세션 동안 유지됨

### 비밀번호 변경하려면

`admin.html` 파일 수정:
```javascript
const ADMIN_PASSWORD = 'artifact2025'; // ← 여기를 원하는 비밀번호로 변경
```
